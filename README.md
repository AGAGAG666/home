## 快速开始

```bash
# 克隆项目
git clone https://github.com/AGAGAG666/AG_home.git
cd AG_home

# 安装依赖
npm install && npm install react-syntax-highlighter @types/react-syntax-highlighter

# 启动开发服务器
npm run dev
```

在浏览器中打开 [loveag.dpdns.org](https://loveag.dpdns.org) 查看效果。

- **图标**: Lucide React
- **主题**: next-themes
